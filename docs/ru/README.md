# Гостевая книга OpenSauced

Эта гостевая книга предназначена для тех, кто прошёл курс "[Введение в Open Source](https://intro.opensauced.pizza/#/)" от [OpenSauced](https://app.opensauced.pizza/) чтобы сделать свой первый вклад в проекты с открытым исходным кодом.

## Подготовка к работе

Подробную инструкцию, как добавить себя в нашу гостевую книгу, можно найти в разделе "[Перейдем к практике](https://intro.opensauced.pizza/#/05-how-to-contribute-to-open-source?id=let39s-get-practical)" в нашем  курсе "Введение в Open Source".

## Решение конфликтов слияния

Если Вы столкнётесь с конфликтами слияния при внесении изменений в этот репозиторий, прочитайте раздел "[Конфликты слияния в репозитории Гостевой книги](https://intro.opensauced.pizza/#/05-how-to-contribute-to-open-source?id=merge-conflicts-in-the-guestbook-repository)" в нашем курсе "Введение в Open Source".

## Что дальше?

Вы можете добавить информацию о Вашем вкладе в этот проект на Вашу страницу [Избранного](https://app.opensauced.pizza/feed) в OpenSauced! Прочитайте нашу документацию, чтобы узнать больше о [функциях раздела Избранное](https://docs.opensauced.pizza/features/highlights/).

Хотите ещё попрактиковаться? Загляните в наш репозиторий [пиццевселенной](https://github.com/open-sauced/pizza-verse) и присоединяйтесь к нашему празднику пиццы!🍕

## 🤝 Внесите свой вклад

Все участники обязаны соблюдать наши [Нормы поведения](https://github.com/open-sauced/.github/blob/main/CODE_OF_CONDUCT.md). Пожалуйста, следуйте нашим [Рекомендациям](CONTRIBUTING.md) при внесении вклада.

- **Улучшение курса**: Если Вы хотите улучшить курс "Введение в Open Source" от OpenSauced, пожалуйста, создайте issue в нашем [репозитории введения](https://github.com/open-sauced/intro), опираясь на [Рекомендации по внесению вклада](https://github.com/open-sauced/intro/blob/main/CONTRIBUTING.md).

- **Переводы**: Если Вы хотите перевести README и другие необходимые части этого репозитория, ознакомьтесь с нашими [Рекомендациями по переводу](i18n-guidelines.md).
